§ R-01 · Active Research · Feb 2026 – Present

Hybrid Neural Accelerator

A reconfigurable LUT / logarithmic compute engine for edge-AI inference — bridging Python-trained neural networks to a custom IP block on Xilinx Artix-7, with PPA optimisation as a first-class design constraint.

TypeFPGA hardware accelerator
TargetArtix-7 XC7A200T
ToolingVivado HLS · Vitis · AXI4
SubmissionGLSVLSI 2026 (LBS)

§ 01 — Overview
The question

Edge inference is bottlenecked by the cost of multipliers. As precision scales down — INT8, INT4, ternary, binary — the relative overhead of memory traffic and control logic dominates the silicon budget. This project asks whether a hybrid compute primitive — switching between LUT-based and logarithmic-domain arithmetic on a per-layer basis — can claw back area, power and timing simultaneously.

The accelerator is being co-designed with a sensitivity-driven quantisation pipeline (the subject of the GLSVLSI 2026 submission) so that layers most tolerant of approximation are mapped to the cheapest compute path automatically.

§ 02 — Architecture
Sub-systems

SYS-01
Compute Engine
Reconfigurable LUT / log datapath written in Vivado HLS. Switches mode based on layer descriptor; INT8 → INT4 paths share storage.
SYS-02
RTL Backend
HLS-synthesised RTL placed on Artix-7 XC7A200T. Focus on timing closure, BRAM/LUT/DSP balance, and clock-domain hygiene.
SYS-03
AXI4 Interconnect
Memory-mapped AXI4 between the custom IP and an embedded ARM core; DMA-driven weight streaming and burst-aligned activations.
SYS-04
Quantisation Pipeline
Python-side sensitivity analysis selects per-layer precision; descriptors are baked into the bitstream metadata at synthesis time.
SYS-05
Verification
Bit-accurate C++ model paired with cocotb testbenches. Coverage-driven regression on a representative DNN slice.
SYS-06
PPA Telemetry
Vivado reports parsed into a sweep dashboard so design-space exploration over precision × topology is reproducible.

§ 03 — Status
Where it stands

+ HLS COMPUTE PRIMITIVE
First-pass LUT / log datapath synthesised; INT8 inference correct against reference model. Logarithmic-mode error bounded under the planned quantisation envelope.
+ RTL PROTOTYPE
Block placed on Artix-7 XC7A200T with timing closure at the target frequency for the first compute tile. Resource utilisation tracked across precision configurations.
+ AXI4 INTEGRATION
Memory-mapped path from ARM core to custom IP in place; DMA-driven activation streaming validated on small workloads.
+ PUBLICATION
Companion paper on the quantisation methodology — “Quantization-Aware Training for Efficient Edge DNN Deployment” — approved for GLSVLSI 2026 Late Breaking Session.
